자주 묻는 질문

Q.왜 단순 자동화 툴이 아닌 의사결정 엔진이 필요한가요?

업무 효율화만으로는 비즈니스의 치명적인 리스크를 막을 수 없기 때문입니다. Aegis는 단 한 번의 잘못된 경영 판단으로 인한 막대한 비용 손실을 사전에 차단하는 '지적 보완 장치'입니다.

Q.경영 및 기업 데이터 유출 위험은 없나요?

기본적으로 모든 대화와 데이터는 철저히 암호화되어 관리됩니다. 최고 보안이 필요한 경우 Enterprise 티어의 전용망 구성(VPC) 및 On-Premise 연동 옵션을 통해 완벽한 폐쇄성을 보장합니다.

Q.어떤 리더에게 가장 유용한가요?

신사업, 대규모 투자 등 중대한 판단을 앞둔 C레벨 및 대표이사는 물론, 조언을 구할 곳 없이 모든 의사결정을 홀로 감당해야 하는 자영업자 및 소상공인 분들에게도 리스크를 사전에 방어해 주는 든든한 필수 파트너가 됩니다.

Q.AI API 키는 어떻게 발급받나요?

처음 설정이 걱정되신다면 걱정 마세요. 에서 Gemini API 키 발급부터 앱 연결까지 단계별로 안내해 드립니다. 5분이면 완료됩니다.

Gemini를 추천하는 이유

Google AI Studio에서 무료로 API 키를 발급받을 수 있습니다. 별도의 결제 정보 없이 바로 시작 가능합니다.

1

Google AI Studio 접속

아래 링크를 클릭해서 Google AI Studio에 접속합니다. 구글 계정으로 로그인하세요.

aistudio.google.com 바로가기
2

API 키 만들기

로그인 후 왼쪽 메뉴에서 『Get API key』 버튼을 클릭합니다.

Google AI Studio
Home
My Library
Get API key← 여기!

그 다음 『Create API key』 버튼을 클릭하면 키가 생성됩니다.

3

API 키 복사

생성된 API 키가 화면에 표시됩니다. 아래와 같이 생긴 긴 문자열입니다.

AIzaSyXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXX

API 키는 비밀번호와 같습니다. 다른 사람에게 공유하거나 인터넷에 올리지 마세요.

4

AEGIS 앱에 입력

앱을 실행하고 아래 순서로 진행하세요.

①앱 상단 오른쪽 ⚙️ 설정 아이콘 클릭
②『AI 설정』 또는 『API 키』 탭 선택
③AI 제공사로 『Google Gemini』 선택
④복사한 API 키를 입력란에 붙여넣기 후 『저장』

저장 후 『연결 테스트』 버튼을 누르면 정상 작동 여부를 바로 확인할 수 있습니다.
